# ERP System for ISO Certification Company

This project is an **Enterprise Resource Planning (ERP) system** developed for an **ISO Certification company**.  
The system is designed to manage certification workflows, financial operations, and accreditation processes in a centralized platform.

## üì¶ Core Modules

- **Quotation**  
  Create and manage quotations for certification services.

- **Invoice**  
  Generate invoices, manage billing, and track payments.

- **Certification Management**  
  Handle certification lifecycles, audit schedules, and certificate issuance.

- **Accreditation Management**  
  Manage accreditation records, compliance requirements, and validity tracking.

- **Client Management**  
  Maintain client profiles, certification history, and documentation.

- **Other Supporting Modules**  
  Additional administrative and operational modules to support daily business activities.
